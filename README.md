# 交易数据抓取工具 v1.0.6

一个专业的期货平仓数据抓取和分析工具。

## 功能特性

- ✅ **合并模式**：建仓提醒 + 平仓提醒数据自动匹配
- ✅ **完整数据**：13字段（开仓时间/方向/商品/开仓点位/止损/止盈/平仓时间/平仓点位/获利点数/盈亏金额/老师/状态）
- ✅ **盈亏计算**：支持小纳指/微纳指/恒指/美原油/美黄金/黄金/小德指/德指
- ✅ **老师汇总**：胜率/盈亏比/总盈亏统计
- ✅ **筛选功能**：日期/老师/方向/商品多维度筛选
- ✅ **点位编辑**：开仓点位/平仓点位/获利点数/盈亏金额可手动修改
- ✅ **自动刷新**：1/3/5/10分钟可选
- ✅ **CSV导出**：一键导出数据
- ✅ **密码保护**：锁屏保护敏感数据

## 界面风格

暗夜高端风 - 专业交易终端风格
- 深蓝黑渐变背景
