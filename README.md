# 交易数据抓取工具 v1.0.6

一个基于 Electron 的期货交易数据抓取和分析工具。

## 功能特点

- ✅ 合并模式：建仓提醒 + 平仓提醒数据自动匹配
- ✅ 13字段完整数据：开仓时间/方向/商品/开仓点位/止损/止盈/平仓时间/平仓点位/获利点数/盈亏金额/老师/状态
- ✅ 盈亏金额自动计算（支持小纳指/微纳指/恒指/美原油/美黄金/黄金/小德指/德指）
- ✅ 老师盈亏汇总（胜率/盈亏比/总盈亏）
- ✅ 筛选功能（日期/老师/方向/商品）
- ✅ 点位编辑功能
- ✅ 自动刷新（1/3/5/10分钟可选）
- ✅ CSV导出
- ✅ 密码锁屏保护
- ✅ 自动更新检测

## 技术栈

- 前端：HTML/CSS/JavaScript
- 后端：Node.js + Express
- 桌面框架：Electron
- 打包工具：electron-builder
- 自动更新：electron-updater
